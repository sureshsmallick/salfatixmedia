Sidebar or Icon

This procedure shows you how to toggle the display of either application icons or the right sidebar.

Overview

The theme-toggle-right-sidebar-min-width theme variable specifies the minimum width (in pixels) that the browser window must be to display the application icons. When the browser width is less than the width specified by the style, the application icons are replaced by the sidebar icon, which can be clicked to display the right sidebar. The style is defined in the theme.scss.json file.

TIP: The higher the number, the more likely the application icons will be replaced by a right sidebar icon!

Edit the value of the theme variable

Open <GBC_PROJECT_DIR>/customization/<customization_project_dir>/theme.scss.json file in a text editor.

Specify the number of pixels to trigger the replacement of the application icons with the right sidebar:

  • If the width of the browser is less than the width of the theme-toggle-right-sidebar-min-width, the right sidebar icon displays. Clicking the icon opens the right sidebar.
  • If the width of the browser is greater than the width of the theme-toggle-right-sidebar-min-width, the application icons display.

Set the theme-toggle-right-sidebar-min-width style to a low number of pixels.

In this example, the right sidebar is disabled as it is set to 0px.

// in theme.scss.json
{
  ...
  "theme-toggle-right-sidebar-min-width" : "0px",
  ...
}

Rebuild using grunt.

$ grunt --customization=<customization_project_dir>

Test

Test the sidebar works as expected. Open your application and expand / contract the browser width.

TIP: You may need to use CTRL + F5 to clear the browser cache before you see your changes.
