Tab background image

This procedure shows you how to change the default empty tabbed background image.

Save your image

Save your background image with the name "emptytabbed.png".

Place the image in your <GBC_PROJECT_DIR>/customization/<customization_project_dir>/resources/img directory.

NOTE: The default empty tabbed background image is named emptytabbed.png. It is referenced in <GBC_PROJECT_DIR>/src/index.html. This page is currently not customizable, therefore you need to name your empty tabbed background image "emptytabbed.png".

Rebuild using grunt.

$ grunt --customization=<customization_project_dir>

Test

Test the empty tabbed background image is displayed as expected.

TIP: You may need to use CTRL + F5 to clear the browser cache before you see your changes.
