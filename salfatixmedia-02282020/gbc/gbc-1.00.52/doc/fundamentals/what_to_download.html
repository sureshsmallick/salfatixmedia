Downloads

When you access the GBC packages for download, you should find two packages: the runtime package and the project package.

Runtime package

The runtime package is a fully-compiled package of the Genero Browser Client. You can download this package and deploy it on to your production system as-is. It provides a default GBC, giving you the default look-and-feel and theme options. You cannot modify the runtime package.

Project package

The project package is the package you must download if you plan to do any customization at all. It gives you the project files in the directory structure that you will need to start your customization efforts.
