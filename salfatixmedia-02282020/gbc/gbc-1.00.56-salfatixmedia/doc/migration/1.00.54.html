GBC 1.00.54

This page provides information about new features, as well as changes you need to plan for when migrating your customization to this version.

For a detailed list of GBC 1.00.54 changes, please refer to the list of fixes on our issue tracker.

Collapsing groups at first launch

GBC supports two new values for the initiallyCollapsed Group presentation style attribute:
always Always collapse a GROUP (if it has the collapsible style) at first launch, ignoring the stored settings and any previous state.
never Never collapse a GROUP (if it has the collapsible style) at first launch, ignoring the stored settings and any previous state.

For more information, see the Genero Business Development Language User Guide.
