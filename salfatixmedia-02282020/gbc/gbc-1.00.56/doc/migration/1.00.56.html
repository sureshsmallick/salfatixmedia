GBC 1.00.56

This page provides information about new features, as well as changes you need to plan for when migrating your customization to this version.

For a detailed list of GBC 1.00.56 changes, please refer to the list of fixes on our issue tracker.

Group title now supports empty title

You can now set a title with only space(s) " " for a GROUP :

 GROUP mygroup (text = "")

ContextMenu improvement

Image action and action updates are now properly managed in context menu.

iOS : smooth scrolling effect

When scrolling on iOS touch device you will get a smooth scrolling effect without focus conflicts if moving finger over input widgets such as DATEEDIT.

Performance improvement

Performance have been improved on several cases during layout processing.
