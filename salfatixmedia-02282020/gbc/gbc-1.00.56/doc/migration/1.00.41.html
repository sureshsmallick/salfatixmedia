GBC 1.00.41

GBC 1.00.41 was released December 1st, 2017.

This page provides information about new features, as well as changes you need to take care of when migrating your customization to this version.

Context Menu

Right-clicking on an application, or on a widget in an application, displays a context menu showing the list of possible actions as in GDC.#

This behavior is driven by a customization variable: theme-disable-context-menu.
