Search the documentation

To search the GBC documentation, you must open it within Genero Studio.

  1. Open Genero Studio.
  2. Open the Studio help.
  3. Create a filter to display only GBC documentation:
    • Select Edit > Preferences and the Filters tab.
    • Create a new filter called "GBC Only". In the Attributes list for this filter, exclude all products except GBC.
    • Click OK to close the Preferences dialog.
  4. In the Filtered by list, select the GBC Only filter.
  5. Select the Search tab and enter your search terms.

Before you leave the help system, remember to revert the Filtered by criteria to Unfiltered. Otherwise, all future searches will be on the GBC documentation only.
