Migrate GDC to GBC

This section explains what actions you need to take to migrate to the Genero Browser Client from other Genero front-end clients.

GDC to GBC

There are features supported by the Genero Desktop Client (GDC) that are not fully supported by the Genero Browser Client (GBC). Read about these features and limitations prior to migrating your application from GDC to GBC. Understanding and planning for the limitations will not only help you determine whether the GBC is a suitable front-end for your application, but it will also greatly assist your migration efforts.

NOTE: This section does not provide a step-by-step migration path, as each application can have issues unique to itself. Instead, this section gives you tips and recommendations based on experience.
